# 爱莉希雅风格网站

这是一个基于崩坏三爱莉希雅角色设计的梦幻风格网站，具有完整的响应式设计和丰富的交互体验。

## 🎨 设计特色

- **色彩方案**: 粉色(#FFB6C1)、浅蓝(#87CEEB)、淡紫(#D8BFD8)
- **视觉元素**: 花朵、水晶、羽毛、星空背景
- **动画效果**: 花瓣飘落、星光闪烁、浮动元素
- **字体选择**: 优雅的手写字体和圆润字体
- **响应式设计**: 完美适配桌面端和移动端

## 🚀 快速开始

### 本地预览
1. **直接打开**：在浏览器中打开 `index.html` 文件
2. **本地服务器**（推荐）：
   ```bash
   # 使用Python（端口8000）
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   
   # 访问地址：http://localhost:8000
   ```

### 移动端测试
- **开发者工具**：在Chrome/Firefox中按F12，使用设备模拟器
- **真实设备**：在同一网络下访问服务器IP地址
- **二维码扫描**：使用手机扫描本地服务器生成的二维码

## 🎯 自定义指南

### 图片资源替换

#### 壁纸轮播图片
替换 `img/` 文件夹中的壁纸文件：
- `爱莉壁纸1.png` - `爱莉壁纸5.jpg`
- 支持JPG、PNG格式，建议尺寸：1920x1080

#### 表情包图片
替换 `img/` 文件夹中的表情包：
- `表情包1.gif` - `表情包4.gif`
- 支持GIF动态图片，建议尺寸：80x80

#### 动态壁纸视频
替换 `video/` 文件夹中的视频：
- `【崩坏3】爱莉希雅-爱神使者-4K.mp4`
- 支持MP4格式，建议分辨率：1080p或4K

### 样式自定义

#### 颜色主题修改
在 `style.css` 中修改主要颜色：
```css
/* 主要配色方案 */
:root {
    --primary-pink: #FFB6C1;    /* 主要粉色 */
    --primary-blue: #87CEEB;    /* 主要蓝色 */
    --primary-purple: #D8BFD8;  /* 主要紫色 */
    --text-color: #ffffff;      /* 文字颜色 */
    --background: rgba(255, 255, 255, 0.05); /* 背景透明度 */
}
```

#### 动画效果调整
```css
/* 修改动画时长 */
.carousel-slide {
    transition: opacity 1.5s ease-in-out; /* 壁纸切换时间 */
}

/* 修改浮动动画 */
@keyframes float {
    0%, 100% { transform: translateY(0px) rotate(0deg); }
    50% { transform: translateY(-20px) rotate(180deg); }
}
```

### 内容编辑
直接编辑 `index.html` 文件中的以下内容：
- 页面标题和描述
- 个人资料信息（角色类型、武器、属性等）
- 关于爱莉希雅的介绍文字
- 故事时间线的对话内容
- 表情包对应的标题和描述

## 🛠️ 技术栈

### 前端技术
- **HTML5** - 语义化标签、视频播放、响应式设计
- **CSS3** - Flexbox布局、Grid布局、CSS动画、媒体查询

### 响应式设计
- **断点设计**：768px（平板）、480px（手机）
- **布局方式**：Flexbox + CSS Grid
- **适配策略**：移动优先（Mobile First）

### 动画效果
- **CSS动画**：关键帧动画、过渡效果
- **JavaScript动画**：轮播切换、滚动动画
- **性能优化**：硬件加速、will-change属性
